DESKRIPSI PROYEK
Rafflestore adalah sebuah sistem informasi berbasis web yang dikembangkan untuk mendukung pengelolaan operasional pada UMKM Sandal. Sistem ini dirancang untuk mengintegrasikan berbagai proses bisnis dalam satu platform, sehingga dapat meningkatkan efisiensi, akurasi data, dan kemudahan pengelolaan usaha.
Fungsi utama sistem Rafflestore meliputi:
•	Manajemen akun pengguna dan karyawan (Owner, Admin, Karyawan)
•	Pengelolaan data master produk (jenis, warna, ukuran/size)
•	Manajemen data produk dan stok barang
•	Pencatatan produk masuk dan produk keluar
•	Pencatatan absensi karyawan harian
•	Penyajian laporan keuangan secara otomatis berdasarkan transaksi
Dalam proses pengembangannya, sistem Rafflestore telah melalui tahap Unit Testing menggunakan metode Black Box Testing untuk memastikan setiap modul berfungsi sesuai dengan kebutuhan fungsional dan siap digunakan dalam operasional usaha sehari-hari.

TECH STACK
Berikut ini adalah tools dan teknologi yang digunakan dalam pengembangan dan pengujian sistem Rafflestore:
•	Visual Studio Code (VS Code)
Digunakan sebagai code editor utama dalam pengembangan sistem.
•	Laravel
Framework backend berbasis PHP yang digunakan untuk membangun logika aplikasi, manajemen data, dan validasi sistem.
•	MySQL
Digunakan sebagai sistem manajemen basis data untuk menyimpan data pengguna, produk, stok, absensi, dan transaksi.
•	HTML, CSS, JavaScript
Digunakan untuk membangun antarmuka pengguna (User Interface) agar sistem mudah digunakan.
•	Web Browser (Google Chrome / Mozilla Firefox)
Digunakan sebagai media pengujian sistem berbasis web.
•	Black Box Testing
Metode pengujian perangkat lunak yang digunakan untuk melakukan unit testing berdasarkan input dan output tanpa melihat struktur kode program.
